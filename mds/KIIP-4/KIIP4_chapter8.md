📌 의미 (Meaning)
다른 사람에게 들은 내용을 상대방에게 확인하듯이 물을 때 사용한다.

🇬🇧 Used when confirming with someone the content you heard from another person — "I heard that...? Is that right?"
🇳🇵 अरूबाट सुनेको कुरा अगाडिको व्यक्तिसँग पुष्टि गर्न सोध्दा प्रयोग गरिन्छ — "मैले सुनेको... हो र?"
📊 Form Chart (형태)
Type	Ending	Example
동사 (받침 O)	-는다면서요?	먹다 → 먹는다면서요? / 듣다 → 듣는다면서요?
동사 (받침 X)	-ㄴ다면서요?	가다 → 간다면서요? / 기다리다 → 기다린다면서요? / 살다 → 산다면서요?
형용사	-다면서요?	비싸다 → 비싸다면서요? / 끝났다 → 끝났다면서요?
명사 + 이다	명사(이)라면서요?	학생이라면서요? / 의사라면서요?

🔵 문법 2: 동-을 겸 동-을 겸
📌 의미 (Meaning)
어떤 행동을 하는 목적이 두 가지 이상임을 나타낼 때 사용한다.

🇬🇧 Used to express that the purpose of doing an action is two or more things — "both to ... and to ..."
🇳🇵 कुनै काम गर्ने उद्देश्य दुई वा सोभन्दा बढी हुँदा प्रयोग गरिन्छ — "...पनि गर्न र ...पनि गर्न"
📊 Form Chart (형태)
Type	Ending	Example
동사 (받침 O)	-을 겸	먹다 → 먹을 겸 / 걷다 → 걸을 겸 / 찾아보다 → 찾아볼 겸
동사 (받침 X)	-ㄹ 겸	가다 → 갈 겸 / 만나다 → 만날 겸 / 만들다 → 만들 겸
장을 보다 → 장을 볼 겸		
📌 Note: Used as "동-을 겸 동-을 겸 (해서)" — usually two purposes listed together.
