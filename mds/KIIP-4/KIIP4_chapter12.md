💡 핵심 정리 (Key Summary)

🔹 선거 종류 3대 축: 대선 (대통령) → 총선 (국회 의원) → 지방 선거 (시장·도지사·교육감 등).

🔹 선거 과정 순서 (자연스러운 흐름):
선거 운동 → 투표소 방문 → 신분 확인 → 투표용지 받기 → 기표소에서 투표 → 투표함에 넣기 → 개표 → 당선

🔹 혼동 주의 – 지지율 vs 득표율:
・지지율 = 선거 전, 여론조사에서의 지지 (support before)
・득표율 = 개표 후, 실제 받은 표의 비율 (actual votes after)

🔹 혼동 주의 – 투표소 vs 기표소:
・투표소 = 투표하는 장소 전체 (whole polling station)
・기표소 = 그 안에서 도장 찍는 작은 칸막이 (the small booth inside)

📖 KIIP 4 · 12과 선거와 투표 · p.159 (문법 1 – 동-나 보다 / 형-은가 보다)

이 페이지는 추측(짐작)을 나타내는 문법을 배우는 페이지예요.

🔵 문법 설명 (Grammar Explanation)

동-나 보다, 형-은가 보다 → 말하는 사람의 추측을 나타낸다.
🇬🇧 Expresses the speaker's guess/inference (based on evidence).
🇳🇵 बोल्ने व्यक्तिको अनुमान (देखेको आधारमा) जनाउँछ।

이전에 배운 -는 것 같다와 비슷하지만, -나 보다/-은가 보다는 눈앞의 근거를 보고 추측할 때 더 자연스러워요.

형태 (Forms):

품사	결합	예시
동사	-나 보다	읽다 → 읽나 보다 / 지지하다 → 지지하나 보다
ㄹ탈락 동사	-나 보다	만들다 → 만드나 보다
형용사 (받침 O)	-은가 보다	좋다 → 좋은가 보다 / 높다 → 높은가 보다
형용사 (받침 X)	-ㄴ가 보다	성실하다 → 성실한가 보다
명사	명+인가 보다	후보자 → 후보자인가 보다

💡 핵심 정리 (Key Summary)

🔹 핵심 규칙:
・동사 → -나 보다 (읽나 보다, 하나 보다)
・형용사(받침O) → -은가 보다 / (받침X) → -ㄴ가 보다 (높은가 보다, 성실한가 보다)
・명사 → 명+인가 보다 (후보자인가 보다)

🔹 추측의 근거가 중요: 이 문법은 눈앞의 상황이나 증거를 보고 추측할 때 써요. 그래서 "-는 것을 보니 / -는 걸 보면 / -는데요" 같은 근거 표현과 자주 같이 나와요.

🔹 -는 것 같다 vs -나 보다:
・-는 것 같다 = 일반적 추측, 자기 생각 포함 가능
・-나 보다 = 근거를 보고 하는 추측 (내 일에는 안 씀. ❌ 나는 배고프나 봐요)

・대통령 = 5년 (혼자만 다름!)
・국회 의원·지자체장·지방 의원 = 모두 4년

🔹 문법 복습: 질문 3번 샘플에서 이번 과 문법 명이야말로("선거야말로")를 활용했어요.

🔹 문화 비교: 한국과 네팔 모두 선거 4원칙을 공유하지만, 정부 형태는 달라요 (한국=대통령제, 네팔=의원내각제 + 대통령). 말하기·쓰기에서 비교하면 풍부한 내용이 돼요.
